# 🌦️ Weather Tracker

A simple Python command-line application that fetches live weather information for any city using the Open-Meteo API.

## Features

- 🌍 Search weather by city name
- 🌡️ Display current temperature
- 💨 Show wind speed
- 🧭 Show wind direction
- ⏰ Display weather update time
- ❌ Handle invalid city names and network errors

## Technologies Used

- Python
- Requests Library
- Open-Meteo API
- Object-Oriented Programming (OOP)

## Installation

1. Clone the repository
2. Install the required package

```bash
pip install requests
```

## Run the Project

```bash
python weather_tracker.py
```
